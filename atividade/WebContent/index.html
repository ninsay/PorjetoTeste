Tutorial de como usar o Git-hub

Conectando a IDE Eclipse

  • Projeto - (IDE Eclipse)
  • Repositório local - (pasta local)
  • Repositório remoto - (Git-hub)

No Eclipse

  • Window - Preference
  • Quando abrir a tela, procurar por "Default repository folder"
  • Clique em "Browse" e crie uma pasta chamada "git" (onde os projetos ficarão guardados)
  • Aperte "Apply and close"

Abrindo a guia do Git-Hub

  • Window - Show View - Other
  • Na tela "Show View", pesquise "git" e escolha a opção "Git Stating"
  • Aperte Open

Abrindo o Repositório

  • Window - Show View - Other
  • Pesquise "git" e escolha a opção "git repositories"
  • Aperte Open

    • Criando um novo Repositório no Git-Hub

  • Acesse o site Git-Hub - Opção "New"
  • Escolha o nome do repositório - "Create repository"
  • Copie o código Token gerado
  • Window - Preference - Configuration - Add Entry
  • Em "Key", digite "user.password" e em "value" cole seu Token
  • Add - Apply and close

    • Ligando a IDE com a pasta local

  • Aperte com o botão direito encima do projeto
  • Team - Share project
  • Na janela "Share project", escreva o nome do seu projeto
  • Finish

    • Enviando o projeto para o Git-Hub

  • Aperte com o botão direito encima do projeto
  • Team - Add to index
  • Ou, clique nas setas "+" logo abaixo das guias
  • Aperte em "Commit and push"
  • Na janela seguinte, em "URL", cole seu código Token
  • Preview - Preview - Push

    • Seu projeto foi devidamente enviado
